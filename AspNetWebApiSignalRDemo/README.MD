```
git init
git remote add -f origin https://github.com/jaliyaudagedara/Blog-Post-Samples.git
git config core.sparseCheckout true
echo "AspNetWebApiSignalRDemo/*" >> .git/info/sparse-checkout
git pull origin master
```
